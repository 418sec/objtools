collapseToDotted

(
  • obj
  • [includeRedundantLevels]
  • [stopAtArrays]
)
Object static

Defined in lib/index.js:102

Given an object, converts it into a one-level-deep object where the keys are dot-separated paths and the values are the values at those paths.

Parameters:

  • obj Object
    • The object to convert
  • [includeRedundantLevels] Boolean optional
    • If set to true, the returned object also includes keys for internal objects. By default, an object such as { foo: { bar: "baz"} } will be converted into { "foo.bar": "baz" }. If includeRedundantLevels is set, it will instead be converted into { "foo": { bar: "baz" }, "foo.bar": "baz" } .
  • [stopAtArrays] Boolean optional
    • If set to true, the collapsing function will not descend into arrays.

Returns:

Object:
  • The result

Example:

By default, an object such as { foo: [ "bar", "baz" ] } is converted into { "foo.0": "bar", "foo.1": "baz" }. If stopAtArrays is set, this will instead be converted into { "foo": [ "bar", "baz" ] } .
